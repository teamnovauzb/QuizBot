Informatika Savol-Javoblar (1-102)
1. 'Find and Replace' qayerda joylashgan?
→ 'Home' menyusi ichida, yoki Ctrl + H yordamida chaqiriladi.
2. 'Mail Merge' funksiyasi qaysi vazifani bajaradi?
→ Bir xil hujjatga har xil ma'lumotlarni joylab, ommaviy yuborish imkonini beradi.
3. 'Page Layout' menyusi nima uchun kerak?
→ Sahifa formatini (chekka, o'lcham, orientatsiya, ustunlar) sozlash uchun.
4. Animatsiya nima uchun kerak?
→ Slayddagi elementlarga harakat berib, kontentni jonlantirish uchun.
5. Autosum tugmasi nima uchun kerak?
→ Excel'da tez yig'indi (SUM) funksiyasini ishlatish uchun.
6. BIOS ning to'liq nomi nima?
→ Basic Input/Output System.
7. BIOS qanday tizimga tegishli?
→ Kompyuterning firmware (past darajali dasturiy ta'minot) qismiga.
8. Bulutli texnologiyalar nima imkoniyat beradi?
→ Internet orqali saqlash, ishlash va ilovalardan foydalanish imkonini yaratadi.
9. Bulutli texnologiyalarda faylni saqlash uchun qaysi xizmat ishlatiladi?
→ Google Drive, Dropbox, OneDrive kabilar.
10. Chipset kompyuterda nima vazifani bajaradi?
→ Ona plata ichida komponentlar o'rtasidagi ma'lumot almashinuvini boshqaradi.
11. Cloud backup tizimi qanday ishlaydi?
→ Fayllarni avtomatik tarzda masofaviy serverga yuklaydi va zaxira qiladi.
12. Cloud Computing qanday asosiy afzallikni beradi?
→ Resurslarni osongina kengaytirish, xarajatni kamaytirish va samaradorlikni oshirish.
13. Cloud texnologiyada 'scalability' nimani anglatadi?
→ Resurslarni (CPU, RAM, disk) dinamik ravishda oshirish yoki kamaytirish.
14. CMOS batareya kompyuterning qaysi komponentini saqlab turadi?
→ BIOS sozlamalari va tizim soatini.
15. Conditional Formatting qanday vazifani bajaradi?
→ Ma'lumotga shart asosida rang, shrift va fon formatini belgilash.
16. Ctrl + B tugmalari nima vazifani bajaradi?
→ Matnni qalin (bold) qiladi.
17. Ctrl + C va Ctrl + V nima vazifani bajaradi?
→ Ma'lumotni nusxalash (copy) va joylashtirish (paste).
18. Ctrl + S tugmalari nima uchun ishlatiladi?
→ Hujjatni saqlash uchun.
19. Ctrl + Z qanday vazifa bajaradi?
→ Oxirgi amalni bekor qilish (undo).
20. Data Validation nima vazifani bajaradi?
→ Ma'lumot kiritishda formalar va qiymatlarni cheklash uchun.
21. Direct Memory Access (DMA) kontrolleri nima vazifani bajaradi?
→ CPU'siz I/O qurilmalar bilan xotira orasida to'g'ridan-to'g'ri ma'lumot uzatadi.
22. Dropbox qanday xizmat?
→ Bulutli saqlash va fayl sinxronizatsiyasi xizmati.
23. ECC (Error-Correcting Code) xotiraning asosiy vazifasi nima?
→ Xotira xatolarini aniqlash va tuzatish orqali ishonchlilikni oshiradi.
24. Elektron pochta (email) nima uchun kerak?
→ Internet orqali yozma xabarlar, hujjat, fayllarni yuborish va qabul qilish uchun.
25. Excel dasturi asosan nima uchun ishlatiladi?
→ Jadval ko'rinishda ma'lumotni tahlil qilish va hisob-kitoblar uchun.
26. Excel dasturining asosiy vazifasi:
→ Elektron jadvallar orqali hisobotlar, grafiklar, tahlillar yaratish.
27. Excel fayllarining kengaytmasi nima?
→ .xlsx
28. Excelda formulalarni boshlash uchun nima yoziladi?
→ Ekval (=) belgisi.
29. Excelda formulani kiritish qanday belgidan boshlanadi?
→ = belgisi.
30. Excelda jadvalini qanday saqlash mumkin?
→ Ctrl + S yoki Fayl → Saqlash menyusi orqali.
31. Excelda katak deb nimaga aytiladi?
→ Excel jadvalidagi bitta maydon (cell).
32. Excelda katakdagi ma'lumotni nusxalash uchun qaysi tugmalar ishlatiladi?
→ Ctrl + C va Ctrl + V
33. Excelda ma'lumotlarni filtrlash qayerda amalga oshiriladi?
→ 'Data' menyusidagi Filter opsiyasida.
34. Excelda makroslar (Macros) nima uchun kerak?
→ Takrorlanuvchi vazifalarni avtomatlashtirish uchun.
35. Excelda 'Sheet' nima?
→ Excel hujjatidagi alohida varaqlar (tabs).
36. Excelda ustun yoki qatorni yashirish qayerdan amalga oshiriladi?
→ Ustun nomi yoki qator raqamiga o'ng tugma bosib, 'Hide' ni tanlash.
37. Excelda ustunlar qanday belgilanadi?
→ A, B, C … formatda.
38. Fayl tizimining vazifasi nima?
→ Ma'lumotlarni diskda tartib bilan saqlash va boshqarish.
39. Google Docs dasturining vazifasi:
→ Onlayn matn hujjatlarini yaratish va birgalikda tahrirlash.
40. Google Docs qanday turdagi fayllarni qo'llab-quvvatlaydi?
→ .doc, .docx, .odt, .rtf, .txt, .pdf, .xls va boshqalar.
41. Google Docs qayerda ishlaydi?
→ Internet va brauzer orqali (cloud'da).
42. Google Drive nima?
→ Google'ning bulutli saqlash xizmati.
43. Google Drive qanday xizmat turi?
→ SaaS (Software as a Service) va IaaS (Infrastructure as a Service).
44. Google Drive'da faylni boshqalar bilan qanday ulash mumkin?
→ Fayl ustiga o'ng tugma → Share, link orqali.
45. Google Drive'da faylni o'chirish uchun nima qilinadi?
→ Faylni tanlab Delete yoki Trash'ga ko'chiriladi.
46. Google hisobiga ega bo'lish uchun nima kerak?
→ Gmail manzili yoki mavjud email bilan hisob ochish.
47. GPU (Graphics Processing Unit) ning vazifasi nima?
→ Grafikalarni grafik kartaga ishlash va vizual tasvirni yaratish.
48. Hujjatni qanday saqlash mumkin?
→ Ctrl + S yoki menyuda 'Save' opsiyasini tanlash orqali.
49. Hyper-Threading (Simultaneous Multithreading) texnologiyasi nima uchun mo'ljallangan?
→ CPU ichida bir nechta threadlarni bir vaqtda ishlashini ta'minlash.
50. IaaS va PaaS farqi nimada?
→ IaaS – infratuzilmani (server, saqlash) beradi; PaaS – dastur platformasini, joylashtirish va rivojlantirish muhiti.
51. IF funksiyasi nima qiladi?
→ Shartga ko'ra farqli natijalar qaytaradi (misol: =IF(A1>0,"Musbat","Manfiy")).
52. INDEX va MATCH funksiyalari qachon ishlatiladi?
→ Jadval ichida mos qiymat qidirish va chiqarish uchun (VLOOKUP alternativ sifatida).
53. Internet nima?
→ Global kompyuter tarmog'i bo'lib, ma'lumot almashuv imkonini beradi.
54. Internet orqali fayllarni saqlashning asosiy afzalligi:
→ Istalgan joydan va qurilmadan kirish mumkin.
55. Interrupt (uzilish) nima vazifa bajaradi?
→ CPU operatsiyasini to'xtatib, ustuvor vazifani bajarish signalini beradi.
56. Kernel nima?
→ Operatsion tizimning yadro qismi bo'lib, apparat resurslarini boshqaradi.
57. Kompyuterning asosiy qismlaridan biri bu:
→ CPU, RAM, disk, ona plata, GPU, PSU.
58. Kursorni harakatlantirish uchun qaysi qurilma ishlatiladi?
→ Sichqoncha yoki klaviaturaning strelka tugmalari.
59. L1 cache qayerda joylashgan?
→ To'g'ridan to'g'ri CPU ichida joylashadi.
60. Makroslar (Macros) nima vazifani bajaradi?
→ Takrorlanuvchi vazifalarni yozib olib, bir klikda bajaradi.
61. Microsoft Word dasturi nima uchun ishlatiladi?
→ Matnli hujjatlarni yaratish, tahrirlash va formatlash uchun.
62. Modem vazifasi nima?
→ Raqamli signalni analogga aylantirib internetga ulanishni ta'minlaydi.
63. Monitor nima uchun ishlatiladi?
→ Vizual ma'lumotni ekranga chiqarish uchun.
64. Motherboard (ona plata) nima vazifani bajaradi?
→ Kompyuter komponentlarini ulash va ularning o'rtasida aloqa ta'minlash uchun.
65. MS Word dasturida asosan nima bilan ishlanadi?
→ Matn tahrirlash, formatlash, sahifalash.
66. MS Wordda sahifa raqamini qayerdan kiritiladi?
→ Insert → Page Number menyusidan.
67. Multidasturlash tizimi nimani anglatadi?
→ Bir vaqtning o'zida bir nechta dasturlarni bajarish imkonini beradi.
68. Named Range nima?
→ Excelda bir yoki bir nechta katakka nom berish imkoniyati.
69. Narration funksiyasi nima vazifa bajaradi?
→ PowerPoint'da ovozli izohlar qo'shish uchun.
70. Operatsion tizim nima?
→ Windows, macOS, Linux.
71. Operatsion tizim nima uchun kerak?
→ Foydalanuvchi va apparat o'rtasidagi vositachi sifatida ishlaydi.
72. Operatsion tizim nima?
→ Kompyuter resurslarini boshqaruvchi dasturiy ta'minot.
73. Operatsion tizimlar turlariga misol keltiring.
→ Desktop (Windows), Server (Windows Server), Mobil (Android/iOS), Real-Time (RTOS).
74. Operatsion tizimning asosiy vazifalaridan biri nima?
→ Resurslarni (CPU, xotira, disk, tarmoq) boshqarish.
75. PCI Express x16 sloti qancha PCIe kanal (lane) bilan ishlaydi?
→ 16 ta lane bilan.
76. Pivot Table nima uchun ishlatiladi?
→ Katta hajmdagi ma'lumotni tahlil qilish va ko'rgazmali hisobot berish uchun.
77. PowerPoint dasturi nima uchun ishlatiladi?
→ Vizual imkoniyatli taqdimotlar yaratish uchun.
78. PowerPoint dasturining asosiy vazifasi nima?
→ Slaydlar orqali kontentni taqdim etish va vizual tarzda ko'rsatish.
79. PowerPoint fayllarining kengaytmasi nima?
→ .pptx
80. PowerPointda animatsiya nima uchun kerak?
→ Elementlarga harakat berish va slaydni jonlantirish uchun.
81. PowerPointda hyperlink qanday qo'shiladi?
→ Insert → Link menyusidan.
82. PowerPointda rasmni slaydga qanday qo'shiladi?
→ Insert → Picture menyusidan.
83. PowerPointdagi slaydlar orasida avtomatik o'tish qayerda sozlanadi?
→ Transitions menyusida.
84. PowerPointdagi har bir sahifa nima deb ataladi?
→ Slayd.
85. Printer qaysi turga kiradi?
→ Chiqish (output) qurilmasi.
86. PSU (Power Supply Unit) qaysi vazifani bajaradi?
→ Kompyuterni elektr bilan ta'minlaydi, kuchlanmalarni uyg'unlaydi.
87. Qanday funksiyalar asosiy matematik hisob-kitoblar uchun ishlatiladi?
→ +, -, *, /, SUM, AVERAGE, MIN, MAX.
88. Qanday qilib hujjatni parol bilan himoyalash mumkin?
→ Word'da File → Protect Document → Encrypt with Password.
89. Qanday tugma yordamida taqdimotni ishga tushirish mumkin?
→ F5 tugmasi.
90. Qattiq disk nima vazifani bajaradi?
→ Doimiy ravishda ma'lumot saqlash.
91. Qaysi menyuda 'Bold', 'Italic', 'Underline' opsiyalari mavjud?
→ Home menyusining Font guruhida.
92. Qaysi menyuda slayd dizayni tanlanadi?
→ Design menyusida.
93. Qaysi menyudan grafikka kirmoq mumkin?
→ Insert → Chart menyusidan.
94. Qaysi protokol veb sahifalarni yuklash uchun ishlatiladi?
→ HTTP yoki HTTPS.
95. Quyidagilardan qaysi biri kiritish qurilmasi?
→ Klaviatura, sichqoncha, skaner.
96. Quyidagilardan qaysi biri operatsion tizim hisoblanadi?
→ Windows, Linux, macOS.
97. Quyidagi qurilmalarning qaysi biri yordamida Internetga ulanish mumkin?
→ Modem, router, mobil modem, Wi-Fi adapter.
98. Real-time OS qanday tizim?
→ Vaqt jihatdan sezgir va tez javob berishi kerak bo'lgan tizim (masalan: avtomobil boshqaruv).
99. Sahifa raqamini qanday qo'shish mumkin?
→ Word'da Insert → Page Number opsiyasidan.
100. Sichqoncha qaysi qurilmalar toifasiga kiradi?
→ Kiritish (input) qurilmalariga.
101. Skanner nima?
→ Qog'oz hujjat yoki surat raqamli formatga aylantirurvchi qurilma.
102. Slaydga matn qanday qo'shiladi?
→ Insert → Text Box yordamida.
